# **App Name**: QuizCraft AI

## Core Features:

- Gmail Authentication: Secure user authentication via Gmail, with OTP verification.
- Onboarding Form: Collects user information (name, profession, age) upon initial login to personalize the experience.
- Gemini API Key Input: Allows users to input and manage their personal Gemini API key for quiz generation.
- AI Quiz Generation: Generates quizzes based on a topic, question count, and difficulty level using the Gemini API; this feature uses the tool to validate generated questions with given source.
- Manual Quiz Creation: Provides an interface for users to create quizzes manually by adding questions, options, and correct answers.
- Quiz Download: Enables users to download quizzes in PDF, JSON, or text formats for offline use.
- Quiz History & Progress Tracking: Stores and displays past quizzes, scores, and progress via a visual graph, showing learning trends and improvements.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evocative of study and concentration.
- Background color: Light lavender (#F3E5F5), which gives a light background that remains on the same hue.
- Accent color: Berry (#E91E63), providing good visual contrast.
- Headline font: 'Space Grotesk' sans-serif, appropriate for headlines, paired with 'Inter' sans-serif for body text.
- Use flat, line-style icons related to education and learning.
- Clean and intuitive layout with clear sections for quiz creation, history, and progress.
- Subtle animations for loading and transitions to enhance user experience.
